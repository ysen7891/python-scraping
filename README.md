## ■ 簡易スクレイピング

## ■ 概要
簡易的なスクレイピングを行ってCSVを出力するプログラムです。

## ■ 主な機能

config.pyにスクレイピング対象のサイトを記入してmain.pyで起動しCSVに出力します。

## ■ 使用技術

- Python 3.13.9
- 使用ライブラリ→BeautifulSoup,logging,time,requests,csv

## ■ 工夫した点

処理ごとにpyファイルを分割し、config.pyに設定を記述するだけでスクレイピングが可能になるように記述しました。

## ■ 作成にあたって注意したこと

スクレイピング対象のサイトはスクレイピング練習用に設定されているサイトを利用しました。その他time.sleep処理を入れることにより対象サイトのサーバーに負荷が少ないようにしました。
